RNA-seq Study Project

In this work, we performed differential RNA expression analysis to investigate changes in expression patterns before and during fermentation in Saccharomyces cerevisiae. We demonstrated changes in levels of expression of 3193 genes, affecting 40 biological processes in GO terms and analyzed the correspondence between gene expression level and function of translated protein during the transition to fermentation.

  • Report: Google Docs
